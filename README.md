# Agent Skill Papers

We treat agent skills as reusable capability units between foundation-model competence and situated task execution. This list follows the survey taxonomy, covering how skills are represented, formed, evolved, orchestrated, evaluated, governed, and applied. Papers that span multiple themes are placed under their most useful primary category.

## News
